Documentation

This website acts as a documention for the Schema.org external extension: netcomp.

The structure and design of the documentation follow the one of Schema.org for more consistency with the core vocabulary. You will also find links to the core vocabulary everytime the extension use or extends one of their classes or properties.

This extension includes 22 new classes, 145 new properties and 10 extended properties.

Note that this documentation as well as the RDFA file have been auto-generated by a python script that you can find in the github repository. It might be useful if you are looking to create your own extension of Schema.org.
